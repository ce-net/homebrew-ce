# homebrew-ce

Homebrew tap for [CE](https://github.com/ce-net/ce) — peer-to-peer compute mesh and economy.

## Install

```bash
brew tap ce-net/ce
brew install ce
```

Or in one step:

```bash
brew install ce-net/ce/ce
```

## Updating

```bash
brew update && brew upgrade ce
```

## Source

Formula tracks pre-built binaries from [ce-net/ce releases](https://github.com/ce-net/ce/releases).
SHA256 values are updated by `packaging/scripts/update-sha256.sh` in the main repo on each release.
